--- notes: | For common use in a toolbars or other groups place as many buttons as desired inside a container with the `.btn-bar` class. Compress the space between such buttons with `.btn-bar--tight`. Implement dividers by adding `.btn--left-divider` or `.btn--right-divider` on the appropriate element. Button bars can also implement classes that affect all the child buttons in order to simplifythe markup. These can be user individually and in combination: * `.btn-bar--small` to make all contained buttons be small buttons. * `.btn-bar--icons` makes all contained buttons icon buttons. * `.btn-bar--clear` makes all contained buttons clear buttons. * `.btn-bar--tight` compresses the spacing between buttons to conserve overall space. ---

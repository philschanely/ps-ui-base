--- notes: | Our icon library implements the font awesome icon library with easier handles Icons can be included through `.icon` classes applied preferrably to `i`, `b`, or `span` tags. Consider using `title` attribute to caption the meaning of the icon in context. Font awesome provides a solid version of most icons. Enable this variation using `.icon--solid`. Icons can also be resized using `.icon--sm`, `.icon--lg`, `.icon--xl`, `.icon--xxl`, or `.icon--xxxl`. ---
